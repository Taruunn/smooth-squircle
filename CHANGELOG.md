# Changelog

All notable changes to this project will be documented in this file.



## [0.1.0] - 2026-01-21

### Added
- Initial release
- `Squircle` React component with full prop support
- `useSquircle` hook for advanced usage
- Core path generation functions (framework-agnostic)
- Responsive radius support (mobile/desktop)
- SVG border overlay for squircle-shaped borders
- Smart fallback for unsupported browsers (Shopify mobile, older Safari)
- Full TypeScript support
- Tree-shakeable exports
